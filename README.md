# 🎴 StudyFlow — فلاش كاردز الفاينل + خطة مذاكرة

تطبيق محلي خفيف لمراجعة أسئلة اختيارات (٤ خيارات) لشباتر **الفاينل فقط** في موادك الأربع، بتكرار متباعد لا نهائي + خطة مذاكرة بعدّاد تنازلي. **محمول بالكامل** — الأسئلة مدموجة داخل التطبيق، يشتغل على البيسي والجوال بدون إنترنت.

## ▶️ التشغيل
- **محلياً:** دبل-كليك على `index.html`.
- **على الجوال (بدون رفع):** انسخ كامل مجلد `Study` للجوال وافتح `index.html` بالمتصفّح.
- **مرفوع على الإنترنت:** شوف قسم «الرفع» تحت.

## 📚 المحتوى — ٥٩٨ سؤال فريد (بعد إزالة التكرار المفهومي)
| المادة | الشباتر | أسئلة |
|---|---|---|
| رسم الحاسب · CCCS-325 | 5 · 5.5 · 7 · 8 · 9 · 11 | 121 |
| مستودعات البيانات · CCDS-221 | 5 · 6 · 7 · 8 · 9 | 146 |
| الذكاء الاصطناعي · CCAI-221 | 7 · 8 · 9 · 10 · 11 · 12 · 13 | 175 |
| تفاعل الإنسان والحاسب · CCSW-225 | 3 · 4 · 5 · 6 · 7 · 8 · 11 | 156 |

**المصادر:** بنوك أسئلتك (📘 بنك الكلية) + فلاش كاردز محاضراتك (🎓 محاضرة) + أسئلة مولّدة (✨ إضافي). كل فكرة مرة وحدة (التعريف والتطبيق يبقون منفصلين). زر **«بنك الأسئلة الرسمي فقط»** يخفي الإضافية.

## ✨ المزايا
- **تكرار متباعد لا نهائي:** صح → مستوى أعلى (١→٥) وتباعد · غلط → مستوى ١ وتكرار قريب · خلط الخيارات.
- **📅 خطة المذاكرة:** عدّاد تنازلي لكل اختبار + خطة كل يوم. لكل شابتر: ذاكر **📖 محاضرة + 📝 ملخص + 💡 كونسبت** أول، ثم **🃏 فلاش كاردز**. علّم الخطوات وانت تخلّص.
- **🔔 تنبيهات حقيقية:** ملف `studyflow.ics` — استورده مرة في تقويم الجوال/الآيفون = تنبيهات مذاكرة يومية + قبل كل اختبار.
- **⚠️ راجع غلطاتي:** شابتر / مادة / كل المواد.
- **📝 الناقص (يدوي):** لكل شابتر — ارسم / معادلات / تمارين / تنبيهات.
- **📊 نِسَب الإتقان والإنجاز** + عرض رياضي للمعادلات.
- **⚙️ إعدادات:** نسخة احتياطية للتقدّم (تنزيل/استرجاع) · قفل بكلمة سر اختياري · تصفير.
- **📲 PWA:** قابل للتثبيت على الجوال (يشتغل أوفلاين كتطبيق) عند رفعه على الإنترنت.

## ⌨️ اختصارات
`1`-`4` / `A`-`D` = إجابة · `Enter`/`مسافة` = التالي · `Esc` = رجوع

## ☁️ الرفع على الإنترنت (بدون دومين خاص)
الملفات كلها في هذا المجلد جاهزة. اختر طريقة:

### الأسهل — Vercel (سحب وإفلات)
1. سجّل دخول/حساب مجاني على vercel.com (عبر GitHub أو Google).
2. من اللوحة: **Add New → Project → Deploy** ثم اسحب مجلد `Study` كامل، أو ثبّت `npm i -g vercel` وقل لي أرفعه.
3. يطلع رابط مثل `studyflow.vercel.app`.

### GitHub Pages
1. حساب GitHub مجاني → New repository (مثلاً `studyflow`).
2. ارفع **كل ملفات مجلد Study** (Add file → Upload files → سحب وإفلات → Commit).
3. Settings → Pages → Source: `main` / root → Save.
4. الرابط: `https://<username>.github.io/studyflow`.

> كله مجاني. الرابط عام (أي أحد يفتحه يشوف الأسئلة) — لو تبي خصوصية، فعّل **القفل بكلمة سر** من ⚙️ الإعدادات. تقدّمك محفوظ بمتصفّحك، فأي أحد يفتح الرابط ما يأثر عليك.

## 🗂️ الملفات
`index.html` (التطبيق+الأسئلة) · `manifest.webmanifest` + `sw.js` + `icon-512.png` (PWA) · `studyflow.ics` (التقويم) · نسخة بيانات قابلة للتعديل في الفولت: `…/Flashcards/data/*.js`.

## 🆕 مزايا جديدة
- 🎯 **امتحان تجريبي (Mock):** زر 🎯 بالأعلى — اختر مادة/نطاق/عدد + مؤقّت، بدون تصحيح فوري، ونتيجة + مراجعة بالنهاية.
- 🧮 **مسائل وحلول:** زر 🧮 على الشابتر — ١٣٥ مسألة حسابية/تطبيقية من بنوكك «اكشف الحل» وقيّم نفسك.
- 🔥 **هدف يومي + ستريك:** في خطة اليوم (عدّل الهدف من ⚙️).
- ⭐ **تمييز بنجمة:** نجمة على كل بطاقة + زر «⭐ المميّزة» لمراجعتها.
- 📉 **أضعف شباترك** · 🔎 **بحث** في كل الأسئلة والمسائل.
- 🔊 **صوت/اهتزاز** عند الإجابة + اختيار **وقت تذكير المذاكرة** وتنزيل تقويم محدّث (من ⚙️).

## 🔄 المزامنة التلقائية بين أجهزتك (GitHub · مشفّرة)
تمشي عبر **حسابك في GitHub** (بدون طرف ثالث)، وبياناتك **مشفّرة** في Gist خاص (ولا GitHub يقدر يقرأها):
1. روح `github.com/settings/tokens` ← Generate new token ← أعطه صلاحية **gist** فقط.
2. في StudyFlow: ⚙️ ← المزامنة ← الصق التوكن ← **تفعيل** (يُحفظ في جهازك فقط).
3. على الجهاز الثاني: نفس التوكن = مزامنة تلقائية. زر «اختبر الاتصال» يتأكد إن التوكن صالح.
